# 주니어 VOCA 단어장 (remaster)

능률 주니어 VOCA 기본 **1200단어(DAY 01~60)**를 아이가 학원 DAY 시험에 맞춰 외우도록 만든 오프라인 웹앱(PWA)입니다.

> 단어 데이터는 기존 [`jr-voca-app`](https://github.com/yeunsu0918/jr-voca-app)에서 가져왔고, 앱 자체는 새로 작성했습니다. 원본 리포는 건드리지 않습니다.

## 주요 기능

- **플래시카드 학습** — 카드 뒤집기, 영어 **발음(TTS)**, 어려운 단어 ⭐ 별표
- **4가지 퀴즈** — 영어→뜻, 뜻→영어, 듣고 뜻 고르기, 스펠링 입력
- **오답 · 간격반복(라이트너 5단계)** — 틀린 단어를 익힘 단계별로 다시 출제해 시험 점수를 올림
- **진도 · 정답률 · 연속학습(스트릭)** — DAY별 마스터리 시각화
- **여러 학습자 프로필** — 남매가 각자 진도 관리
- **오프라인 지원 · 폰 홈화면 설치(PWA)** · 다크모드 · 글자 크기 조절

## 실행

빌드 과정이 없습니다. 정적 파일을 아무 서버로나 띄우면 됩니다.

```bash
# 예: python
python3 -m http.server 4000
# 또는 bun
bunx serve .
```

브라우저에서 `http://localhost:4000` 접속. 폰에서 열고 "홈 화면에 추가"하면 앱처럼 설치됩니다.

## 구조

```
index.html              앱 셸
css/styles.css          스타일 (라이트/다크, 모바일 우선)
js/data.js              단어 데이터 (60일 × 20단어, 자동 생성)
js/store.js             상태·저장·라이트너 SRS·스트릭
js/quiz.js              문제 생성 (4모드, 오답 보기 생성)
js/tts.js               발음 (Web Speech)
js/app.js               화면 라우터·UI 컨트롤러
sw.js                   오프라인 캐시
manifest.webmanifest    PWA 설정
```

진도는 브라우저 `localStorage`에 학습자별로 저장됩니다.
